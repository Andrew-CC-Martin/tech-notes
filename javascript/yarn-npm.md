# yarn

## config

### Set config

`$ yarn config set <config> <setting>`

#### proxies

- http
- `$ yarn config set proxy <address>`
- https
- `$ yarn config set https-proxy <address>`

#### ssl

`$ yarn config set strict-ssl false`

# npm

## log in

### verify if logged in

- `$ npm whoami` - will return username if logged in, else will error
